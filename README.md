## Overview

The **TBB Arena** is the official competition arena developed for **TheBoxingBots**.

This repository provides the complete set of design resources required to manufacture and assemble the arena. Whether you're a customer, educational institution, robotics club, maker, or event organizer, you'll find the official engineering files, manufacturing drawings, and visual references needed to build the arena.

---

## Repository Structure

```text
.
├── TBB Arena Model SS
├── TBB Arena Parts Dimensions
├── TBB Arena Rendering
└── TBB Arena Size
```

| Folder                         | Description                                                               |
| ------------------------------ | ------------------------------------------------------------------------- |
| **TBB Arena Model SS**         | Reference model images of the complete arena.                             |
| **TBB Arena Parts Dimensions** | Engineering drawings, manufacturing PDFs, and STEP model for fabrication. |
| **TBB Arena Rendering**        | High-quality rendered images of the completed arena.                      |
| **TBB Arena Size**             | Arena size and dimensional reference images.                              |
